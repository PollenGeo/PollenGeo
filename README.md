## About the Project
 
The **PollenGEO** project, led by palynologist **Carlos Jaramillo** at STRI, brings together more than 30 researchers and students to digitize the entire Smithsonian palynological collection — one of the largest in the world, comprising over 18,000 tropical plant species.

## 🔬 Omero\_Pollen
 
> Tools, scripts and pipelines for **image management and processing** within the PollenGEO digitization workflow.
 
| Repository | Description |
|---|---|
| 📦 [Omero_Grid](https://github.com/PollenGeo/Omero_Grid) | OMERO grid interface for navigating and organizing microscopy image collections |
| 📦 [OMERO-Project-Dataset-CSV-Manager](https://github.com/PollenGeo/OMERO-Project-Dataset-CSV-Manager) | Automates structured creation and organization of Projects and Datasets in OMERO from a CSV file |
| 📦 [OMERO_Palynology](https://github.com/PollenGeo/OMERO_Palynology) | Comprehensive FAQ and guide for using OMERO.Web in palynological analysis workflows |
| 📦 [Omero-hackaton](https://github.com/PollenGeo/Omero-hackaton) | Hackathon solutions for palynology challenges within OMERO |
| 📦 [Import_CSV](https://github.com/PollenGeo/Import_CSV) | Imports multiple types of ROIs from a CSV file into OMERO images |
| 📦 [Export-everything-in-Omero](https://github.com/PollenGeo/Export-everything-in-Omero) | Exports all user-created ROIs across all accessible groups, datasets and images in OMERO |
| 📦 [Autocomplete_for_Omero_v2.0](https://github.com/PollenGeo/Autocomplete_for_Omero_v2.0) | Browser extension for genus and species name autocomplete in OMERO.Web |
| 📦 [Import-ndpa-annotation-files-into-Omero](https://github.com/PollenGeo/Import-ndpa-annotation-files-into-Omero) | Imports ROIs from NDPA/XML annotation files into OMERO |
| 📦 [OMERO-Image-Dataset-Exporter](https://github.com/PollenGeo/OMERO-Image-Dataset-Exporter) | Retrieves image metadata (dataset, project, and group IDs) from selected OMERO groups and datasets |
| 📦 [Replace-annotations-old-text-to-new-text-](https://github.com/PollenGeo/Replace-annotations-old-text-to-new-text-) | Updates shape comments in OMERO in bulk using a CSV file |
| 📦 [Search-annotations](https://github.com/PollenGeo/Search-annotations) | Searches for species names within image annotations across all OMERO groups |
 
> 🔎 Browse all → **[`#omero-pollen`](https://github.com/PollenGeo?tab=repositories&q=omero-pollen)**

## 🖼️ Bioimages
 
> Repositories for **biological image processing and transformation** of microscopy data from the PollenGEO pipeline.
 
| Repository | Description |
|---|---|
| 📦 [NDPA-Annotation-Transformer](https://github.com/PollenGeo/NDPA-Annotation-Transformer) | Processes paired NDPA files from NDP.view, extracts reference pins and recalculates annotations between crop coordinates |
| 📦 [Convert_czi_to_stl](https://github.com/PollenGeo/Convert_czi_to_stl) | ImageJ macro for converting CZI microscopy files to STL format for 3D visualization |
 
> 🔎 Browse all → **[`#bioimages`](https://github.com/PollenGeo?tab=repositories&q=bioimages)**
