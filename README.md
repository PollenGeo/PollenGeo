## About the Project
 
The **PollenGEO** project, led by palynologist **Carlos Jaramillo** at STRI, brings together more than 30 researchers and students to digitize the entire Smithsonian palynological collection — one of the largest in the world, comprising over 18,000 tropical plant species.

## 🔬 Omero\_Pollen
 
> Tools, scripts and pipelines for **image management and processing** within the PollenGEO digitization workflow.
 
| Repository | Description |
|---|---|
| 📦 `omero-pollen-ingest` | Automated ingestion of microscopy images into OMERO with metadata tagging |
| 📦 `omero-slide-manager` | Tools for organizing and querying the 23,000+ slide dataset in OMERO |
| 📦 `pollen-classifier` | AI model for automated pollen grain identification trained on PollenGEO data |
| 📦 `omero-qc-pipeline` | Quality control workflows: resolution checks, taxonomy validation, noise filtering |
 
> 🔎 Browse all → **[`#omero-pollen`](https://github.com/tuusuario?tab=repositories&q=omero-pollen)**

## 🖼️ Bioimages
 
> Repositories for **biological image analysis** of pollen grains across multiple microscopy modalities.

 
| Repository | Description |
|---|---|
| 📦 `pollen-segmentation` | Grain detection and segmentation from multi-focal microscopy stacks |
| 📦 `confocal-3d-reconstruction` | 3D composite reconstruction from Zeiss LSM 980 confocal image series |
| 📦 `slide-scanner-processing` | Processing pipelines for Hamamatsu Nanozoomer whole-slide scans |
| 📦 `bioimage-annotation` | Annotation tools for taxonomic labeling of pollen grain images |
 
> 🔎 Browse all → **[`#bioimages`](https://github.com/tuusuario?tab=repositories&q=bioimages)**
